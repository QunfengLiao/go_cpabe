# 研究决策：租户管理员新增成员

## 决策 1：按邮箱复用全局账号

邮箱是现有登录身份的稳定唯一标识。命中已有账号时只增加当前租户成员关系和角色，不覆盖密码、用户名、姓名或手机号。

## 决策 2：固定初始密码仅用于新账号

遵循用户给定演示规则使用 `lqf999..`，但只保存 bcrypt 摘要并设置 `must_change_password=true`。响应只在新建成功时返回一次提示；审计和日志不得包含密码。

## 决策 3：只允许 DO/DU

该入口用于普通业务成员，不能授予 `TENANT_ADMIN` 或 `PLATFORM_ADMIN`，避免成员管理权限升级自身权限边界。
